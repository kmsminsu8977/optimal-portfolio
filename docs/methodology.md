# 방법론 노트

## 0. 기준 문서

- `AGENTS.md`: 작업 기준과 품질 기준
- `SOURCE.md`: 참고 경로와 컨텐츠 재구성 기준

## 1. 프로젝트 목적

표본 수익률에서 평균-분산 기준의 단순 최적 가중치를 찾는다.

## 2. 핵심 질문

제약조건이 있는 long-only 포트폴리오에서 Sharpe ratio가 가장 높은 가중치 조합은 무엇인가?

## 3. 입력 데이터

- 파일: `data/sample/asset_return_panel.csv`
- 성격: 구조 검증용 합성 샘플
- 목적: 계산 흐름과 결과 해석을 작은 표본으로 먼저 검증

## 4. 계산 절차

1. 월별 합성 수익률에서 기대수익률과 공분산을 추정한다.
2. 10% 단위 grid search로 long-only, full-investment 제약을 만족하는 가중치를 찾는다.
3. 최적화 절차를 설명하는 기준 구현이다.

## 5. 산출물 해석

`outputs/tables/baseline_results.csv`는 프로젝트별 baseline 결과입니다. 이 표는 최종 연구 결론이 아니라, 이후 실제 데이터 연결과 상세 분석을 시작하기 위한 구조 검증용 산출물입니다.

## 6. 한계

- 합성 데이터 기반이므로 통계적 유의성이나 투자 가능성을 주장하지 않습니다.
- 거래비용, 세금, 슬리피지, 유동성 제약은 일부 프로젝트에서만 단순 반영했습니다.
- 프로젝트별 정교한 모델링은 별도 후속 작업에서 확장해야 합니다.
